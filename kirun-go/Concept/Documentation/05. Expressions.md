# Expressions

## Arithmetic Operators

| Operator | Description      | Example        |
| -------- | ---------------- | -------------- |
| `+`      | Addition         | `5 + 3` → `8`  |
| `-`      | Subtraction      | `5 - 3` → `2`  |
| `*`      | Multiplication   | `5 * 3` → `15` |
| `/`      | Division         | `6 / 2` → `3`  |
| `//`     | Integer Division | `7 // 2` → `3` |
| `%`      | Modulus          | `7 % 3` → `1`  |

## Logical Operators

| Operator | Description           | Example                           |
| -------- | --------------------- | --------------------------------- |
| `and`    | Logical AND           | `true and false` → `false`        |
| `or`     | Logical OR            | `true or false` → `true`          |
| `not`    | Logical NOT           | `not true` → `false`              |
| `<`      | Less Than             | `5 < 10` → `true`                 |
| `<=`     | Less Than or Equal    | `5 <= 5` → `true`                 |
| `>`      | Greater Than          | `10 > 5` → `true`                 |
| `>=`     | Greater Than or Equal | `10 >= 10` → `true`               |
| `=`      | Equal                 | `5 = 5` → `true`                  |
| `!=`     | Not Equal             | `5 != 3` → `true`                 |
| `??`     | Nullish Coalescing    | `null ?? 'default'` → `'default'` |

## Bitwise Operators

| Operator | Description                  | Example          |
| -------- | ---------------------------- | ---------------- |
| `&`      | Bitwise AND                  | `5 & 3` → `1`    |
| `\|`     | Bitwise OR                   | `5 \| 3`→`7`     |
| `^`      | Bitwise XOR                  | `5 ^ 3` → `6`    |
| `~`      | Bitwise Complement           | `~5` → `-6`      |
| `<<`     | Bitwise Left Shift           | `5 << 1` → `10`  |
| `>>`     | Bitwise Right Shift          | `10 >> 1` → `5`  |
| `>>>`    | Bitwise Unsigned Right Shift | `10 >>> 1` → `5` |

## Unary Operators

| Operator | Description              | Example              |
| -------- | ------------------------ | -------------------- |
| `+`      | Unary Plus               | `+5` → `5`           |
| `-`      | Unary Minus              | `-5` → `-5`          |
| `not`    | Unary Logical NOT        | `not true` → `false` |
| `~`      | Unary Bitwise Complement | `~5` → `-6`          |

## Array & Object Operators

| Operator | Description                                  | Example                                 |
| -------- | -------------------------------------------- | --------------------------------------- |
| `[`      | Array Operator (indexing) or Object Operator | `array[0]` or `object["test-property"]` |
| `..`     | Array Range Index Operator                   | `array[1..3]`                           |
| `.`      | Object Operator (property access)            | `object.property`                       |

## Conditional Operators

| Operator | Description                  | Example                       |
| -------- | ---------------------------- | ----------------------------- |
| `?`      | Conditional Ternary Operator | `condition ? value1 : value2` |

## Operator Precedence

Operators are evaluated in the following order (1 = highest precedence):

1. **Highest**: Unary operators (`+`, `-`, `not`, `~`), Array/Object operators (`[`, `.`)
2. Array Range (`..`), Arithmetic (`*`, `/`, `//`, `%`)
3. Arithmetic (`+`, `-`)
4. Bitwise Shift (`<<`, `>>`, `>>>`)
5. Comparison (`<`, `<=`, `>`, `>=`)
6. Equality (`=`, `!=`)
7. Bitwise AND (`&`)
8. Bitwise XOR (`^`)
9. Bitwise OR (`|`)
10. Logical AND (`and`)
11. Logical OR (`or`), Nullish Coalescing (`??`)
12. **Lowest**: Conditional Ternary (`?`)

Array Range operator works on strings too for substring

## Special Features

### String Multiplication

The `*` operator can be used with strings to repeat them:

```
"Hello" * 3  → "HelloHelloHello"
```

### Nesting Expressions

You can use `{{ }}` to nest expressions within expressions:

```
"The result is {{ 5 + 3 }}"  → "The result is 8"
```

### Type Safety

- Object operators have restrictions on which operations can be performed with objects
- Most operators cannot be applied to objects except for equality, logical, and nullish coalescing operations

### Space Wrapping

Some operators like `and`, `or`, `not` are wrapped in spaces during parsing for better readability.

### Constants

Boolean constants are true and false. Numerical constants. String constants. Null constants null, nil and undefined.
